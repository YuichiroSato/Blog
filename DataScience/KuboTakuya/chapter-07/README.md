# 一般化線形混合モデル(GLMM)

## 過分散

データがある分布から得られているとすると、ばらつきが大きすぎるなることがある。
なにかモデルに組み入れられていない変数の影響でそうなっている時は、GLMではデータに合うモデルは作れない。

## 一般化線形混合モデル

分布の積を使って尤度を計算するモデル。

### 例　二項分布と正規分布の混合

p1(yi|b1, b2, ri) 以下のリンク関数を持つ二項分布

logit(qi) = b1 + b2 xi + ri

p2(ri|s) = (1 / sqrt(2πs^2)) exp(- ri^2 / 2s^2)

Li = ∫p1(yi|b1, b2, ri) p2(ri|s) dri

L = Π Li
